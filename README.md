# escape-room game
A browser-based Escape Room game built completely from scratch with HTML, CSS, and JavaScript.
Navigate through 10 challenging levels, dodge traps, and unlock doors to escape the room!
# How To Play
### Movement: Arrow Keys or WASD

<img width="626" height="240" alt="image" src="https://github.com/user-attachments/assets/66b721a2-701c-4e0a-b374-1aa0158ce1df" /> 

- Arrow Up / W → Move up
- Down / S → Move down
- Arrow Left / A → Move left
- Arrow Right / D → Move right
